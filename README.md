# # ✍️ Handwritten Digit Recognizer AI

This is an end-to-end Deep Learning project that recognizes handwritten digits using a Convolutional Neural Network (CNN).

## 🚀 Highlights
- **Framework:** PyTorch
- **Model Training:** 10 Epochs with a Final Loss of 0.0424.
- **Image Processing:** Implemented color inversion to ensure real-world compatibility.
- **Interface:** Real-time web application built with Gradio.

## 🛠️ Files in this Repo
- `mnist_cnn.pth`: Pre-trained model weights (The AI "Brain").
- `Handwritting.ipynb`: Full source code for training and deployment.

## 📖 How to Run
1. Open the `.ipynb` file in Google Colab.
2. Ensure `mnist_cnn.pth` is in the same directory.
